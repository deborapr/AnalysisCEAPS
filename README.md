# AnalysisCEAPS

Análise exploratória e tratamento de dados do Cota para Exercício da Atividade Parlamentar dos Senadores (CEAPS) #7DaysOfCode


## Descrição

Análise exploratória e tratamento de dados do Cota para Exercício da Atividade Parlamentar dos Senadores (CEAPS), desenvolvida durante o desafio 7 Days of Code. O projeto aplica técnicas de ciência de dados para investigar padrões de gastos parlamentares, gerar visualizações, identificar tendências e extrair insights a partir de dados públicos do Senado Federal utilizando Python, pandas e bibliotecas de visualização
